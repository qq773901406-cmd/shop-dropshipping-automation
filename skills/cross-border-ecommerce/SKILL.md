# 跨境电商无货源代发全链路分析

## 商业模式说明

**无货源代发（零库存）**：
- 在 TikTok Shop 上架商品
- 用户下单后，你去低价货源平台下单
- 货源平台直发给用户
- 你赚取差价，无需囤货

**核心竞争力**：选品眼光 + 内容种草能力

---

## 推荐货源平台优先级

| 优先级 | 平台 | 优势 | 适用场景 |
|--------|------|------|---------|
| ⭐⭐⭐ | Temu（美国仓） | 3-7天达，价格低，支持代发 | 主力货源，快时效 |
| ⭐⭐⭐ | 速卖通 | 品类全，可直发美国 | 找不到Temu货源时备选 |
| ⭐⭐ | 1688+转运 | 价格最低 | 有稳定供应商时用 |
| ⭐ | Amazon（抄底） | 1-2天Prime达 | 紧急补单，成本高不建议 |

---

## 执行规则（重要）

每次触发（用户主动调用 或 每日定时 09:00）自动选出 **10 款**商品：
- 爆款小商品（家居/厨房/数码等）：4款
- 女装/男装/童装：2款
- 鞋子：2款
- 包包/配饰：2款

**不询问用户**，直接对 10 款商品依次完整执行全链路分析并输出所有文件。

选品标准：TikTok 话题热度高、代发可行、毛利率 > 50%、内容易拍。

避免重复：跳过 `D:/Product/output/` 中已有文件夹的商品。

全部完成后依次执行：
1. 用 fetch_product_imgs.py + CloakBrowser 下载图片
2. card-generator 生成发布到各个平台图
3. publisher 发布到8个平台：国内（抖音/小红书/B站/微博/知乎）用中文文案，国外（Instagram/Facebook/YouTube）用英文文案
4. 打包zip用 WeComReply 发送到企微群 + 发一条汇总说明（10款利润对比 + 优先上架建议）

## 图片下载规范

图片通过 `D:/Product/fetch_product_imgs.py` 自动下载，**无需手动执行**。

**方案**：CloakBrowser → Google Images → 从 `<script>` 块提取产品 CDN 图片 URL 下载

- 文件命名：`产品_01.jpg` / `产品_02.jpg`（每款2张）
- 禁止：货源图、场景图、说明性 .md 文件
- Amazon 图片 CDN 方案已弃用（bot 检测问题）

## 价格单规范（03_价格单.md）

card-generator 解析格式要求：
- 必须有 `| **建议定价** | **$XX.XX** |` 格式
- 必须有 `| **毛利率** | **XX.X%** |` 格式
- 月收益表格必须有"保守/中等/乐观"三行
- 汇率：1 USD ≈ 7.25 CNY，每次注明当日汇率
- 价格单含美元+人民币双列

## 新手启动 SOP

```
第1天：在 TikTok Shop 开通店铺（美区需要美国手机号/地址）
第2天：在 Temu/速卖通找到货源，确认代发流程
第3天：拍摄第一条产品视频，挂上商品链接
第4天起：每天发1-3条视频，测试哪个方向播放量高
第2周：有单后，在货源平台下单填写客户地址发出
持续：复盘出单视频，放大内容方向，增加SKU
```

## 常见坑位

- **时效问题**：提前在商品描述注明发货时效，避免差评
- **退货处理**：代发模式退货直接退款更划算，别让货回流
- **账号关联**：TikTok Shop 美区运营需要合规账号
- **侵权风险**：避免直接搬运品牌商品，选白牌或自有款
